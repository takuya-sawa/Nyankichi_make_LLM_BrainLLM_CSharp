# BrainLLM - 脳細胞型言語モデル

C# で実装した生物学的に正確なニューロンを使ったLLM。消波ブロック型の複雑な分岐構造を持つニューラルネットワーク。

## 特徴

### 🧠 生物学的に正確なニューロンモデル

**Dendrite（デンドライト - 樹状突起）**
- 複数の入力端子
- シナプス重みを持つ
- STDP（スパイク時間依存可塑性）に対応

**Soma（ソーマ - 細胞体）**
- デンドライト信号の統合
- 膜電位の時間発展
- 閾値超過でスパイク発火

**AxonTerminal（軸索ターミナル）**
- 複数の出力分岐
- シナプス伝達物質の放出
- 他のニューロンへの接続

### 🔗 複雑な分岐構造

```
入力層（埋め込み次元数のニューロン）
  ↓
隠れ層（複数分岐を持つニューロン）
  ├→ ニューロン1 → (複数の軸索ターミナル)
  ├→ ニューロン2 → (複数の軸索ターミナル)
  └→ ニューロンN → (複数の軸索ターミナル)
  ↓
出力層（語彙サイズのニューロン）
```

### 📊 学習メカニズム

- **STDP学習**: スパイク時間依存可塑性
- **Hebbian学習**: 前後シナプス活動に基づく重み更新
- **バックプロップ**: 勾配ベースの最適化

## ビルド・実行

```bash
cd BrainLLM_CSharp
dotnet run -c Release
```

## 実装ファイル

- `Neuron.cs` - ニューロンの実装（デンドライト、ソーマ、軸索）
- `BrainNetwork.cs` - ニューロンネットワーク
- `Program.cs` - トレーニングと推論パイプライン

## パラメータ

| パラメータ | 値 | 説明 |
|-----------|-----|------|
| 埋め込み次元 | 32 | 入力層ニューロン数 |
| 隠れニューロン | 64 | 隠れ層の脳細胞数 |
| 出力ニューロン | 語彙サイズ | 出力層の予測ユニット |
| 学習率 | 0.01 | シナプス可塑性の速度 |

## 消波ブロック型の分岐

このアーキテクチャの「消波ブロック型」の特徴：

- **複数の入力デンドライト** - 複数の信号源から受信
- **複雑な内部統合** - ソーマでの非線形統合
- **複数の出力軸索** - 複数のニューロンへ同時に信号伝達
- **フィードバック接続** - 隠れ層内での再帰的なニューロン間接続

これにより、単純な層状ネットワークより複雑な情報処理が可能になります。
